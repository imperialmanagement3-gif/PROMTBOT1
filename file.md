# 🤖 PromptBot - Generador de Respuestas WhatsApp

Herramienta gratuita para generar respuestas profesionales de WhatsApp para negocios.

## 🚀 Demo en Vivo

[Ver Demo](https://tu-usuario.github.io/promptbot)

## 💰 Modelo de Negocio Freelance

### Cómo Monetizar Este Proyecto

1. **Servicio de Configuración** ($50-150 USD)
   - Personaliza las respuestas para el negocio del cliente
   - Configuras 10-20 respuestas personalizadas
   - Les entregas un archivo con todas sus respuestas

2. **Paquete Mensual** ($30-80 USD/mes)
   - Actualizas promociones semanales
   - Agregas nuevas respuestas según necesiten
   - Soporte por WhatsApp

3. **Implementación con Bot** ($200-500 USD)
   - Conectas con BuilderBot o similar
   - Automatización completa
   - Capacitación al cliente

### Clientes Ideales

- Restaurantes y cafeterías
- Tiendas online pequeñas
- Profesionales independientes
- Clínicas y consultorios
- Academias y escuelas
- Inmobiliarias locales

## 📋 Características

- ✅ 6 tipos de negocios
- ✅ 8 tipos de respuestas por negocio
- ✅ Variables personalizables
- ✅ Historial de respuestas
- ✅ Exportar a archivo
- ✅ Copiar con un clic
- ✅ Sin backend necesario
- ✅ Funciona offline
